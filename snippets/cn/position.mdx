坐标是包含8个整数的数组，格式为`[x1, y1, x2, y2, x3, y3, x4, y4]`

表示一个四边形四个定点坐标，按顺时针排列。
即：

- 左上: (x1, y1)
- 右上: (x2, y2)
- 右下: (x3, y3)
- 左下: (x4, y4)

坐标值为像素的绝对值。坐标系如下:

```
图像坐标系 (原点在左上角)
┌────────────width───────────> x
│(0,0)
│
│
│      (x1,y1) ────────── (x2,y2)
│         │                     │     
height             文本区域        
│         │                     │
│      (x4,y4) ────────── (x3,y3)
│
v
y
```
