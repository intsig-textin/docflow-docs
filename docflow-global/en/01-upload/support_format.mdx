---
title: "File Format Support"
---

Docflow supports the following file formats:

- Images ：JPEG/JPG/PNG
- PDF
- Office ：DOC/DOCX/XLS/XLSX
- OFD
- TXT

## File Limitations

Docflow supported files have the following limitations:

1. Single file size **does not exceed 50MB**
2. Multi-page files **do not exceed 1000 pages**
3. Number of extracted fields **does not exceed 100**
